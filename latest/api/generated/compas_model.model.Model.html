Model

class compas_model.model.Model(name='model', elements=[], copy_elements=False)

Bases: Data

The Model data-structure represents:

  1. flat collection of elements - dict{uuid.uuid4(), compas_model.elements.Element}

  2. hierarchical relationships between elements - compas.datastructures.Tree (compas_model.model.ElementNode or compas_model.model.GroupNode)

  3. abstract linkages (connections between elements and nodes) - compas.datastructures.Graph (str(uuid.uuid4()), str(uuid.uuid4()))

Parameters
namestr, optional

A name or identifier for the model.

elementslist[compas_model.elements.Element], optional

A list of elements to be added to the model.

copy_elementsbool, optional

If True, the compas_model.elements.Element are copied before adding to the model.

Attributes

Name of the model.

elementsdict{uuid.uuid4(), compas_model.elements.Element}

A dictionary of elements.

hierarchycompas.datastructures.Tree

A tree data-structure that stores a tree of elements and group nodes.

interactionscompas.datastructures.Graph

A graph data-structure that stores interactions between elements.

number_of_elementsint

A total count in the compas_model.elements.Element dictionary.

number_of_nodesint

A total count of all compas.datastructures.TreeNode in the hierarchy.

number_of_edgesint

A total count of all edges in the compas.datastructures.Graph.
