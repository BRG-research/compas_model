Element

class compas_model.elements.Element(name=None, id=None, frame=None, geometry_simplified=None, geometry=None, insertion=None, **kwargs)

Bases: Data

Class representing a structural object of an assembly.

Parameters
namestr

e.g., “BLOCK”, “BEAM”, “FRAME” to be consistent majority of names are stored in ELEMENT_NAME class

idlist[int] or int

unique identifier/s , e.g.,0 or [0] or [0, 1] or [1, 5, 9]. one object can have an index and belong to a group/s

frameFrame

local frame of the element there is also a global frame stored as an attribute

geometry_simplifiedlist[Polyline]

minimal geometrical represetation of an object type is a polyline since it can represent: a point, a line or a polyline

geometrylist[any]

a list of geometries used for the element visualization currently supported types: geometry or Mesh

insertionlist[Vector, int]

direction of the element, often defined by a single vector (can be also a sequence) and an index in an insertion sequence

kwargs (dict, optional):

additional keyword arguments can be passed to the element.

Examples

>>> element = Element("BLOCK", 0, Frame.worldXY(), Point(0,0,0),                             Box(                                 frame=Frame([3, 0, 0], [0.866, 0.1, 0.0], [0.5, 0.866, 0.0]),                                 xsize=2, ysize=4, zsize=0.25),                             [Vector(0, 0, 1), 0])
